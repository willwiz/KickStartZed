# Kickstart Zed

This directory contains some configuration files for the settings and keybindings of [Zed](https://zed.dev/), to create a more vscode compatible experience.

## Installation
Clone this repository into your Zed config directory:
Do not forget to back up and rename the configuration folder

## Settings
- Control paste keybindings
- `Ctrl+''`''` for toggle right side located terminal
